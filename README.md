# משחק פלטפורמה מגניב

משחק פלטפורמה שנבנה באמצעות Phaser 3 ו-Vite.

## איך משחקים?
- השתמש בחצים ימינה ושמאלה כדי לזוז
- לחץ על חץ למעלה כדי לקפוץ
- אסוף כוכבים לצבירת נקודות
- היזהר מהפצצות!
- כשנגמר המשחק, לחץ על רווח להתחלה מחדש

## התקנה

```bash
pnpm install
```

## הרצה בסביבת פיתוח

```bash
pnpm dev
```

## בנייה לייצור

```bash
pnpm build
```

## תצוגה מקדימה של גרסת הייצור

```bash
pnpm preview
```

## מה נעשה עד כה
- יצירת משחק פלטפורמה מלא עם:
  - דמות עם אנימציות
  - פלטפורמות לקפיצה
  - כוכבים לאיסוף
  - פצצות כמכשולים
  - מערכת ניקוד
  - אפשרות להתחיל מחדש
